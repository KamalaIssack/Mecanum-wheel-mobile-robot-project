# Hardware Documentation

This directory contains all hardware-related documentation for the Mecanum Wheel Mobile Robot.

## Directory Structure

### `/cad`
CAD files and 3D models for the robot chassis and components.
- Supported formats: STEP, STL, Fusion 360, SolidWorks

### `/schematics`
Electrical schematics and wiring diagrams.
- Circuit diagrams
- PCB layouts (if applicable)
- Wiring diagrams

### `/bom`
Bill of Materials listing all components needed for the build.

### `/datasheets`
Component datasheets for reference.
